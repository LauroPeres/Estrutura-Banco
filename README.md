# Estrutura Bancária

# Descrição
O projeto cria um simulador de operações 
bancárias com o uso da linguagem Java.

# Tecnologias Usadas
- Java
  
# Como Executar
Use o comando "java -cp bin principal.main.Main"
Caso queira fazer alguma modificação, e necessário 
compilar novamente os arquivos .java contidos na
pasta "src".
